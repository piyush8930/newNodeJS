# Hello World Web Service

A simple Node.js web service that serves a Hello World page in the browser.

## Installation

First, install the dependencies:

```bash
npm install
```

## Usage

Start the server:

```bash
npm start
```

Or directly with Node.js:

```bash
node index.js
```

The server will start on port 3000. Open your browser and navigate to:

```
http://localhost:3000
```

You should see a web page displaying "Hello, World!" with some basic styling.

## Features

- Express.js web server
- Serves an HTML page accessible in any browser
- Simple and lightweight
